Projet ACPE
Ce projet comprend un ensemble de scripts Python utilisés pour l'analyse de données de l'Assemblée Parlementaire du Conseil de l’Europe. Ces scripts sont conçus pour traiter des données relatives aux membres, aux groupes politiques, aux pays, et aux adhésions à l'APCE, en vue de produire des analyses et des visualisations pertinentes.
Contenu du Projet
* Tables/ : Ce répertoire contient les fichiers Excel utilisés comme sources de données pour l'analyse. Les tables incluent des informations sur les membres, les groupes, les pays, les adhésions, etc.
* APCE_Phase_1.ipynb : Ce fichier Jupyter Notebook contient le code Python principal pour le traitement des données, l'analyse, et la génération de visualisations.
Dépendances
Ce projet nécessite l'installation des bibliothèques Python suivantes :
* pandas
* numpy
* matplotlib
* seaborn
Pour installer ces dépendances, vous pouvez utiliser pip :
pip install pandas numpy matplotlib seaborn
Utilisation
1. Assurez-vous que toutes les dépendances sont installées.
2. Téléchargez ou clonez ce dépôt sur votre machine.
3. Placez les fichiers de données dans le répertoire Tables/.
4. Ouvrez le fichier APCE_Phase_1.ipynb dans un environnement Jupyter Notebook.
5. Exécutez chaque cellule dans l'ordre pour exécuter le code et générer les analyses et les visualisations.
